산업공학부 대학원 수업 '데이터 기반 학습' 에서 진행한 프로젝트 입니다.
이 프로젝트는 다른 프로젝트들과는 다르게 100% 창작된 코드는 아닙니다.
그래도 많이 변형하고 추가 한것 같습니다. 

수업 조교님이 큰 틀을 만들어주셨고 해당 틀을 변형시키고 가공하여 더 좋은 성능의
머신러닝 기법을 만드는것이 프로젝트의 목표였습니다.

프로젝트가 총 네개 였고 
파이썬을 기반으로 tensor_flow, pandas, scipy, numpy 등의 라이브러리를 합쳐서 만들었습니다.
사용된 머신 러닝 기법으로는 프로젝트 별로 다음과 같습니다.

프로젝트 1. 달러 값 예측(Neural Network)
프로젝트 2. 금 값 예측(Hidden Markov Model)
프로젝트 3. 한국 경마 경기 순위 예측(Support Vector Machine)
프로젝트 4. 강화학습을 통한 주식 투자 알고리즘(Reinforcement Learning)

특히 마지막 프로젝트 4번은 다른 수강생들에 비해 결과가 잘 나왔습니다.
